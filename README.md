# Zero-MAL-Scanner

Free and Open-Source Malware Detection Tool.

Developed for file analysis, signature matching, and threat detection using Python.

### Author

Ahmed Elboushy

### Features

* Malware Signature Detection
* File & Directory Scanning
* Threat Classification
* Scan Reporting
* Desktop GUI

### Requirements

* Python 3.8+

### Dependencies

* tkinter
* hashlib
* threading
* pathlib
* datetime
* json
* re
* os
* sys
* time
* math
